# AuraVerse AI Design Guidelines

## Color Palette

### Background Colors
- **Primary Background**: Pure black (`#000000` / `0 0% 0%`)
- **Card Background**: Very dark gray (`#0D0D0D` / `0 0% 5%`)
- **Secondary Background**: Dark gray (`#1A1A1A` / `0 0% 10%`)

### Text Colors
- **Primary Text**: White (`#FFFFFF` / `0 0% 100%`)
- **Secondary Text**: Light gray (`#B3B3B3` / `0 0% 70%`)
- **Headers**: Purple (`#A78BFA` / `270 75% 65%`)

### Accent Colors
- **Primary Accent**: Purple (`#A78BFA` / `270 75% 65%`)
- **Borders**: Dark gray (`#333333` / `0 0% 20%`)
- **Inputs**: Dark gray (`#262626` / `0 0% 15%`)

### Interactive States
- **Hover**: Purple glow effect
- **Focus**: Purple ring (`270 75% 65%`)
- **Active**: Purple highlight

## Typography

### Font Family
- **Primary**: Inter (sans-serif)
- **Monospace**: Menlo

### Font Sizes
- **Headings**: Use purple color for visibility
- **Body Text**: White for maximum contrast
- **Subtext**: Light gray for hierarchy

## Design Principles

1. **High Contrast**: Black background with white text ensures maximum readability
2. **Purple Accents**: Headers and interactive elements use purple for brand identity
3. **Visibility First**: All elements must be clearly visible against black background
4. **Consistent Spacing**: Use Tailwind spacing utilities for consistency
5. **Purple Glow Effects**: Buttons and cards use subtle purple glow on hover

## Component Styling

### Cards
- Background: Very dark gray (`0 0% 5%`)
- Border: Dark gray for definition
- Text: White for readability
- Headers: Purple

### Buttons
- Primary: Purple background with white text
- Secondary: Dark gray background with white text
- Hover: Purple glow effect

### Inputs
- Background: Dark gray (`0 0% 15%`)
- Text: White
- Border: Light gray for visibility
- Focus: Purple ring

### Navigation
- Background: Pure black
- Text: White
- Active: Purple indicator
- Hover: Purple highlight
